============
Installation
============


Stable release
--------------

To install MultiInfo for Django, run this command in your terminal:

.. code-block:: console

    $ pip install django-multiinfo

This is the preferred method to install MultiInfo for Django, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

You can either clone the public repository:

.. code-block:: console

    $ git clone git@github.com:wooyek/django-multiinfo.git

Or download the download source from `project website`_. Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _project website: https://github.com/wooyek/django-multiinfo
